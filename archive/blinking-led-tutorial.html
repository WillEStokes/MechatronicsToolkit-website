Blinking LED Tutorial

Introduction

In this tutorial, we will learn how to control an LED using a microcontroller. We will be using a simple program to blink an LED on and off.

Materials

  • Microcontroller (e.g., Arduino, Raspberry Pi)
  • LED
  • Resistor (220 ohm)
  • Jumper wires

Wiring

To wire the circuit, connect the positive leg of the LED (the longer leg) to a digital pin on the microcontroller. Connect the negative leg of the LED (the shorter leg) to a 220 ohm resistor. Connect the other end of the resistor to the GND pin on the microcontroller.

Code

To make the LED blink, we will use the built-in `digitalWrite()` and `delay()` functions of the microcontroller. The `digitalWrite()` function sets a digital pin to either HIGH or LOW, and the delay() function pauses the program for a specified amount of time. Here is an example of how you can blink the LED using Arduino code:

                
                    int led = 13;
                    void setup() {
                        pinMode(led, OUTPUT);
                    }
                    void loop() {
                        digitalWrite(led, HIGH);
                        delay(1000);
                        digitalWrite(led, LOW);
                        delay(1000);
                    }

In this example, the LED is connected to pin 13 on the microcontroller. In the setup function, we set the pin as an output using the `pinMode()` function. In the loop function, we use the `digitalWrite()` function to turn the LED on and off, and the `delay()` function to wait for 1 second between each change in state. You should also make sure that the code is compatible with the microcontroller board that you are using.

Conclusion

Congratulations! You have successfully learned how to blink an LED using a microcontroller. Now you can use this knowledge to create more advanced projects, such as a traffic light or a night light. Remember to always use a resistor when connecting an LED to a microcontroller to protect the circuit from damage. Happy building!
